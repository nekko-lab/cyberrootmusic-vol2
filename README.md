## Start dev server

```shell
npm install
npm run dev
```

## Static Site Generator (Node.js)

```shell
# 1. Panda CSSのstyle systemを生成（初回または設定変更時）
npx panda codegen

# 2. クライアントビルド
npm run build.client

# 3. サーバービルド
npm run build.server
```
